🎮 Dobble Game – Spot the Similarity

A logic-based matching game built with Python.

Programming is not about what you know — it’s about what you can figure
out.

🧠 About the Game

Dobble (Spot It!) is a fast-paced observation game.

-   Each card contains 8 symbols
-   Any two cards share exactly one common symbol
-   Players must quickly identify the matching symbol

Example: Card A → Horse, Candle, Star Card B → Candle, Moon, Apple
Common symbol: Candle

------------------------------------------------------------------------

🎯 How It Works

-   Two cards are generated
-   Players compare symbols
-   First to find the single match wins

The challenge: 8 × 8 = 64 possible comparisons — but only one correct
match
